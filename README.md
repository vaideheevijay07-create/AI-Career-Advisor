# AI Career Advisor

## Project Overview

AI Career Advisor is an intelligent career guidance system built using n8n and Groq AI.

The system collects student information through a web form and generates a personalized career assessment report based on the student's skills, branch, interests, and career goals.

## Features

* Personalized Career Recommendations
* Career Readiness Score
* Strength Analysis
* Skill Gap Identification
* Learning Roadmap
* Salary Insights
* Short-Term, Medium-Term, and Long-Term Goals
* Detailed Action Plan
* Professional Report Generation

## Technologies Used

* n8n
* HTML Templates
* JavaScript
* GitHub

## Workflow

Form Submission → AI Agent (Groq AI) → Code Node → Generate HTML Template →Form Ending 

## Project Output

The system generates a detailed career assessment report including:

* Student Information
* Career Matches
* Strengths
* Skill Gaps
* Learning Roadmap
* Recommended Courses
* Career Readiness Score
* Expected Salary Range
* Goals and Action Plan
